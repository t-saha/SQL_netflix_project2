# SQL_netflix_project2
Data analysis of a Netflix dataset using MySQL.

**Dataset** : 
This is a Netflix dataset that contains information about movies and TV shows. Title, director, cast, description, and other data are provided.
https://www.kaggle.com/datasets/shivamb/netflix-shows?resource=download

**Objective**
- Explore and analyze the dataset using SQL.
- Solve a series of questions related to the dataset to find meaningful insights.
